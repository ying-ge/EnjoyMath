## 🔧 技术实现细节

### 核心算法

1. **问题生成算法**
   ```
   输入: 学生陈述 + 学生档案 + 学年级别
   ↓
   AI 提示词生成 (包含模板和上下文)
   ↓
   API 调用 (流式或非流式)
   ↓
   JSON 解析和验证
   ↓
   输出: 结构化问题列表
   ```

2. **分析算法**
   ```
   输入: 学生回答 + 预期概念
   ↓
   AI 分析 (正确性、理解深度、误区)
   ↓
   标签和元数据提取
   ↓
   跟进建议生成
   ↓
   输出: 详细分析报告
   ```

3. **自适应算法**
   ```
   连续正确 ≥ threshold
   ↓
   难度提升 (easy → medium → hard → expert)
   ↓
   重置计数器
   
   连续错误 ≥ threshold
   ↓
   难度降低 (hard → medium → easy)
   ↓
   重置计数器
   ```

4. **复习调度**
   ```
   使用 Ebbinghaus 遗忘曲线
   间隔: [1, 3, 7, 14, 30] 天
   正确时: 按间隔递增复习
   错误时: 重置回第一个间隔
   ```
