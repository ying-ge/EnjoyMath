## 🎯 主要功能

### A. 自动生成苏格拉底问题 🤖

**6种问题类型：**
1. **澄清 (Clarification)** - "你能用自己的话解释吗？"
2. **假设探究 (Probing)** - "为什么你认为这是对的？"
3. **证据 (Evidence)** - "能举个具体的例子吗？"
4. **方法 (Alternative)** - "有没有其他解法？"
5. **推论 (Consequence)** - "如果改变条件会怎样？"
6. **反思 (Metacognition)** - "你是怎么解决的？"

**特点：**
- ✓ 完全自动化，无需手工编写
- ✓ 支持 4 个 AI 服务提供商
- ✓ 问题循序渐进地递增难度
- ✓ 根据学生水平自动调整
- ✓ JSON 结构化输出

### B. 智能分析系统 📊

**分析内容：**
- ✓ 回答的正确性评估（百分比）
- ✓ 理解深度判断（表面 vs 深层）
- ✓ 常见误区自动识别
- ✓ 学生思维方式分析

**输出：**
- ✓ 详细反馈评论

- ✓ 改进方向指导
- ✓ 激励性评语

## 🔧 技术实现

### 核心算法

**问题生成流程：**
```
学生陈述 + 学生档案 + 学年级别
    ↓
AI 提示词生成（包含背景和示例）
    ↓
调用 API（流式或非流式）
    ↓
JSON 解析和验证
    ↓
结构化问题输出
```

**自适应难度算法：**
```
正确计数 ≥ 3 → 难度提升
错误计数 ≥ 3 → 难度降低
保持在 4 个等级之间
```

**复习调度算法：**
```
使用 Ebbinghaus 间隔：[1, 3, 7, 14, 30]天
答对时：按间隔递增
答错时：重置回第一个间隔
```
