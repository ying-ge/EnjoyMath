# AI 辅助苏格拉底提问系统 - 使用指南

## 功能概述

这个增强系统集成了先进的 AI 技术，为初中数学学习提供个性化的苏格拉底式引导。核心功能包括：

### 1. **AI 生成苏格拉底问题** 🤖
- **自动问题生成**: 根据学生的回答，AI 会自动生成启发性问题
- **多层次递进**: 问题从浅层理解逐步深入
- **情境自适应**: 根据学生的水平和学习风格调整问题难度

### 2. **实时学生回答分析** 📊
- **正确性评估**: 评估学生回答的准确程度
- **理解深度检测**: 判断是表面理解还是深层理解
- **误区识别**: 自动识别常见的数学误区
- **跟进建议**: 给出针对性的补充问题

### 3. **个性化学习路径** 🎯
- **前置条件检查**: 确保学生具备必要基础
- **逐步学习方案**: 提供循序渐进的学习步骤
- **关键概念讲解**: 用直观的方式解释核心概念
- **实际应用举例**: 展示数学在现实中的应用

### 4. **学习进度跟踪** 📈
- **学生档案**: 记录每个学生的学习特点
- **掌握情况**: 跟踪已掌握和困难的概念
- **学习分析**: 分析学生的思维方式和学习模式
- **进度报告**: 生成详细的学习报告

### 5. **智能复习计划** 🔄
- **自适应复习**: 根据学生需要生成个性化复习方案
- **分阶段设计**: 从概念回顾到自我检测的完整流程
- **时间规划**: 给出每个阶段的建议耗时
- **评估机制**: 包含自我检测题目

### 6. **主题推荐系统** 💡
- **逻辑学习序列**: 建议最适合的下一个学习主题
- **难度评估**: 评估推荐主题的难度
- **前置知识确认**: 列出需要的前置条件
- **学习时间估算**: 预估学习所需时间

## 使用流程

### 第一步：启动系统
1. 打开 `socratic-dialogue/index.html`
2. 填入 API 配置（支持 OpenAI、Claude、智谱 GLM 等）
3. 选择学生年级和数学主题
4. 点击"测试连接"验证 API
5. 点击"开始对话"进入聊天界面

### 第二步：开始互动
1. **学生提问**: 输入数学相关的问题或陈述
2. **AI 回复**: 系统给出苏格拉底式的引导回复
3. **AI 生成问题**: 自动生成额外的深层问题
4. **继续对话**: 学生根据问题继续思考和回答

### 第三步：系统分析
系统会实时：
- 分析学生的理解程度
- 识别可能的误区
- 跟踪学习进度
- 更新学生档案

## 核心概念

### 苏格拉底式提问的6种类型

1. **澄清问题 (Clarification)**
   - 帮助学生清晰地表达和理解概念
   - 例: "你能用自己的话解释一下这个概念吗？"

2. **假设探究 (Probing Assumptions)**
   - 挑战学生的假设和初步结论
   - 例: "你为什么认为这是对的？"

3. **证据收集 (Evidence)**
   - 要求学生提供支持他们观点的证据
   - 例: "能举个具体的例子吗？"

4. **方法探索 (Alternative)**
   - 激励学生探索不同的解法
   - 例: "有没有其他方法来解决这个问题？"

5. **推论推广 (Consequence)**
   - 帮助学生看到逻辑链条
   - 例: "如果这个条件改变，会导致什么后果？"

6. **反思元认知 (Metacognition)**
   - 促进学生对自己学习过程的反思
   - 例: "回顾一下，你是怎么解决的？"

## 学生档案数据结构

系统为每个学生维护一份档案：

```javascript
{
    gradeLevel: "7",           // 年级
    mathTopic: "代数",         // 当前主题
    learningSpeed: "medium",   // 学习速度
    preferredStyle: "visual",  // 偏好风格
    currentDifficulty: "medium",
    masteredConcepts: [],      // 已掌握的概念
    strugglingConcepts: [],    // 困难的概念
    strengths: [],             // 优势
    weaknesses: []             // 弱点
}
```

## 关键方法说明

### AIAssistant 类

#### 初始化
```javascript
const assistant = new AIAssistant();
await assistant.initialize(config);
```

#### 生成问题
```javascript
const result = await assistant.generateSocraticQuestions(
    "学生的陈述或问题"
);
```

#### 分析回答
```javascript
const analysis = await assistant.analyzeStudentResponse(
    "学生的回答",
    ["期望理解的概念"]
);
```

#### 生成学习路径
```javascript
const path = await assistant.generateLearningPath(
    "当前学习的概念"
);
```

#### 生成复习计划
```javascript
const plan = await assistant.generateReviewPlan(
    ["需要复习的概念"]
);
```

#### 推荐下一主题
```javascript
const recommendation = await assistant.recommendNextTopic(
    "刚学完的主题"
);
```

#### 生成学习报告
```javascript
const report = await assistant.generateLearningReport(
    sessionData
);
```

## 最佳实践

### 1. 引导有效的对话
- 让学生充分表达自己的想法
- 耐心等待学生思考
- 避免急于纠正错误，而是引导发现

### 2. 调整难度
- 根据学生的回答逐步调整问题难度
- 简单问题构建信心，复杂问题促进深思
- 使用"恰好超出能力"的难度（ZPD）

### 3. 鼓励多种表达
- 鼓励用图表、具体例子、语言等多种方式表达
- 接纳不同的解题路径
- 表扬思考过程，而不仅仅是正确答案

### 4. 建立学生档案
- 持续更新学生的学习特点
- 记录优势和困难概念
- 根据档案个性化调整策略

### 5. 定期复习和反思
- 使用复习计划加强已学知识
- 鼓励学生反思学习过程
- 生成报告追踪长期进度

## 常见问题

### Q: AI 生成的问题为什么有时候不太合适？
A: 这通常是因为上下文不足。确保学生的表述足够清晰，AI 才能生成更好的问题。

### Q: 如何处理 API 连接失败？
A: 
1. 检查 API Key 是否正确
2. 确认网络连接正常
3. 检查 API 服务状态
4. 尝试切换 API 提供商

### Q: 系统如何保证学生信息安全？
A: 
- 所有配置保存在本地浏览器
- 可以手动清除所有数据
- 支持导出学习数据

### Q: 可以离线使用吗？
A: 不能。系统需要连接到 AI API 服务才能生成问题和分析。
