# EnjoyMath - 数学学习工具集

探索数学，发现更好的自己。EnjoyMath 为初中生提供自主探索数学能力的工具，包含自适应测试、交互式题库等多种实用工具。

## 🎯 项目简介

EnjoyMath 是一个专为初中生设计的数学探索平台，让你通过自主测试和练习，发现自己的数学能力，找到成长的方向。

## 🛠️ 工具列表

### 1. 自适应数学水平测试
- **路径**: `adaptive-test/index.html`
- **功能**: 基于IRT算法的智能测试系统，根据答题表现动态调整难度
- **特点**:
  - 🎯 **智能高效**：只需完成15-20道题目，就能准确评估能力
  - 📊 自适应难度调整，题目难度随答题情况动态变化
  - 📚 知识点全面覆盖（21个知识点，203道题目库，其中几何题目63道）
  - 💡 提供针对性学习指导和练习题推荐
  - 📈 详细的能力评估报告和知识点掌握度分析
- **适用场景**：探索自己的数学能力，发现强项和提升方向

### 2. 交互式题库
- **路径**: `math-test/题库.html`
- **功能**: 包含全部203道题目，可在网页上直接作答练习
- **特点**:
  - 📖 **完整题库**：包含全部203道精选题目（几何题目63道，包含2024年新教材新增知识点）
  - 🌐 网页直接作答，无需安装
  - ✅ 即时答题反馈，立即知道对错
  - 📝 每道题都有详细解析
  - 📊 支持知识点分析和掌握度统计
- **适用场景**：系统练习所有题目，通过反复练习提升能力

## 🚀 部署到 GitHub Pages

### 快速部署

1. **推送代码到 GitHub**
   ```bash
   git add .
   git commit -m "准备部署到 GitHub Pages"
   git push origin main
   ```

2. **启用 GitHub Pages**
   - 进入仓库 Settings → Pages
   - Source 选择 "GitHub Actions"（推荐）或 "Deploy from a branch"
   - 保存设置

3. **访问网站**
   - 部署完成后，访问 `https://yourusername.github.io/EnjoyMath/`

详细部署说明请查看 [DEPLOY.md](./DEPLOY.md)

## 📖 使用指南

### 自适应数学水平测试

1. 访问 `adaptive-test/index.html`
2. 输入学生ID（可选）或留空
3. 点击"开始探索"
4. 完成15-20道题目
5. 查看详细的能力评估报告

### 交互式题库

1. 访问 `math-test/题库.html`
2. 浏览题目列表
3. 点击选项作答
4. 查看即时反馈和解析
5. 查看知识点掌握度统计

## 🎨 设计特色

- **素气清爽**：灰色科技风格，简洁优雅
- **响应式设计**：完美适配手机、平板、电脑

## 📝 技术栈

- **前端**：纯 HTML + CSS + JavaScript
- **算法**：IRT (Item Response Theory) 2PL 模型
- **存储**：localStorage（客户端存储）
- **部署**：GitHub Pages（静态托管）
