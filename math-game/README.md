# 数学冲刺挑战 (Math Rush)

一个紧张刺激的初中数学记忆理解游戏，基于2025年新版教材设计。

## 🎮 游戏特色

### 🚀 四种游戏模式

1. **⚡ 极限挑战**
   - 60秒限时答题
   - 难度随分数递增
   - 考验反应速度和准确度

2. **🎯 记忆风暴**
   - 90秒记忆公式和概念
   - 重点考察数学基础知识
   - 训练记忆理解能力

3. **🏆 知识竞速**
   - 120秒多知识点答题
   - 随机切换知识点类型
   - 综合能力测试

4. **📚 专题突破**
   - 选择特定知识点练习
   - 90秒深度训练
   - 针对性强化提升

### 🎯 核心玩法

- **紧张刺激**: 限时答题营造紧迫感
- **连击系统**: 连续答对获得额外分数
- **难度递增**: 根据表现动态调整难度
- **即时反馈**: 答题后立即显示对错和解析
- **成就系统**: 解锁各种成就获得满足感

### 📚 教材覆盖

基于2025年新版初中数学教材，涵盖：

- **七年级上**: 有理数、整式、一元一次方程、基础几何
- **七年级下**: 实数、分式、二元一次方程组、不等式、平面几何
- **八年级上**: 二次根式
- **八年级下**: 函数概念、一次函数、反比例函数、三角形、四边形
- **九年级上**: 一元二次方程、二次函数
- **九年级下**: 圆、统计、概率

## 🎮 操作说明

### 键盘操作
- **数字键 1-4**: 选择对应选项
- **空格键**: 暂停/继续游戏
- **ESC**: 返回主菜单

### 触屏操作
- 点击选项按钮选择答案
- 点击暂停按钮暂停游戏

## 🏆 计分规则

### 基础得分
- 基础分数: 10分
- 难度加成: 难度系数 × 基础分数
- 时间奖励: 快速答题获得额外分数

### 连击奖励
- 3连击以上: 连击数 × 5分
- 连击越高，奖励越丰厚

### 模式加成
- 极限挑战: 1.5倍得分
- 记忆风暴: 1.3倍得分
- 知识竞速: 1.2倍得分
- 专题突破: 无额外加成
