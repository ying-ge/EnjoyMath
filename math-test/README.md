# 自适应数学水平测试系统

探索你的数学能力边界。基于IRT（项目反应理论）算法的智能测试系统，根据你的答题表现动态调整难度，帮你发现自己的强项和需要提升的地方。

## 🎯 功能特性

### 自适应数学水平测试系统

- 🎯 **快速探索**：只需完成15-20道题目，就能发现自己的数学能力水平
- 📊 **智能匹配**：基于IRT算法，难度随你的能力自动调整，找到最适合你的题目
- 📚 **全面覆盖**：涵盖初中数学所有主要知识点（19个知识点，103道题目库）
- 🎯 **能力地图**：清晰看到自己在各个知识点的掌握情况
- 📈 **成长报告**：获得个性化的能力分析报告
- 💡 **提升路径**：发现下一步需要重点提升的方向和推荐练习
- 🧮 **公式支持**：完美支持数学公式渲染（LaTeX）
- 📱 **PWA支持**：可安装为移动应用，支持离线使用

### 交互式题库

- 📖 **完整题库**：包含全部103道精选题目
- 🌐 **网页作答**：直接在浏览器中打开即可使用，无需安装
- ✅ **即时反馈**：点击选项立即显示正确与否
- 📝 **详细解析**：每道题都有完整的解析说明
- 📊 **统计分析**：支持知识点掌握度分析和学期表现统计
- 🎯 **推荐练习**：根据答题情况推荐针对性练习题

## 🔍 两个系统的区别

| 特性 | 自适应测试系统 | 交互式题库 |
|------|---------------|-----------|
| **题目数量** | 15-20道（智能选择） | 103道（全部题目） |
| **使用方式** | 需要启动后端和前端服务 | 直接打开HTML文件 |
| **主要用途** | 快速评估能力，获得学习建议 | 系统练习，巩固知识点 |
| **智能程度** | 自适应调整难度 | 固定题目顺序 |
| **评估报告** | 详细的能力评估和指导 | 知识点掌握度统计 |
| **适用场景** | 想了解自己的数学水平 | 想系统学习所有题目 |

## 🚀 快速开始

### 方法1：使用启动脚本（推荐）

#### 启动后端
```bash
./start_backend.sh
```

#### 启动前端
```bash
./start_frontend.sh
```

### 方法2：手动启动

#### 后端启动
```bash
# 安装依赖
pip install -r requirements.txt

# 初始化数据库（首次运行）
cd backend
python3 app.py --init-db

# 启动服务器
python3 app.py
```

后端将在 `http://localhost:5001` 运行。

#### 前端启动
```bash
cd frontend

# 安装依赖（首次运行）
npm install

# 启动开发服务器
npm run dev
```

前端将在 `http://localhost:3000` 运行。

## 🎨 技术栈

- **前端**: React + TypeScript + Vite
- **后端**: Python Flask
- **数据库**: SQLite
- **自适应算法**: IRT (Item Response Theory) 2PL模型
- **公式渲染**: KaTeX
- **UI设计**: Google Material Design

## 📊 题库统计

- **总题目数**: 203 道
- **知识点数**: 21 个
- **分类数**: 6 个主要分类
- **测试题目数**: 15-20 道（自适应）
- **几何题目数**: 63 道（三角形15道、四边形12道、圆14道、平面几何14道、立体几何6道、投影与透视4道）

### 知识点分布

- 数与式：有理数、实数、整式、分式、二次根式（约35道）
  - 包含：简单的代数式（用字母表示数）4道
- 方程与不等式：一元一次方程、二元一次方程组、一元二次方程、不等式（21道）
- 函数：函数概念、一次函数、二次函数、反比例函数（22道）
  - 注意：2024年改版教材中，函数概念和一次函数已从八年级上移至八年级下
- 图形的认识：平面几何（14道）、投影与透视（4道），共18道
- 图形的性质：三角形（15道）、四边形（12道）、圆（14道）、立体几何（6道），共47道
- 图形的认识：平面几何（14道）、投影与透视（4道），共18道
- 统计与概率：统计（17道，包含四分位数、箱线图、抽样）、概率（6道），共23道

### 学期分布（已均衡补充）

- **七年级上**: 21道（有理数、整式、一元一次方程、基础几何）
- **七年级下**: 35道（实数、分式、二元一次方程组、不等式、平面几何）
- **八年级上**: 21道（二次根式，已补充15道）
- **八年级下**: 43道（函数概念、一次函数、反比例函数、三角形、四边形）
- **九年级上**: 36道（一元二次方程18道、二次函数18道，已补充24道）
- **九年级下**: 47道（圆、统计、概率、立体几何、投影与透视）

## 🔧 开发说明

### 添加新题目

编辑 `backend/supplement_questions.py` 或创建新的题目文件，然后运行：

```bash
cd backend
python3 import_all_questions.py
```

### 导出题库

```bash
cd backend
python3 export_questions.py        # 导出Markdown、HTML、文本格式
python3 export_interactive_questions.py  # 导出交互式HTML
```

### 查看统计

```bash
cd backend
python3 question_stats.py          # 题目统计
python3 semester_stats.py          # 学期分布统计
```
