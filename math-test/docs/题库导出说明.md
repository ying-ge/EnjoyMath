## 💡 使用建议

### 给家长查看
**推荐使用 HTML 格式**：
1. 双击 `题库.html` 文件
2. 在浏览器中打开
3. 可以查看所有题目、答案和解析
4. 可以打印（Ctrl+P / Cmd+P）

### 打印使用
**推荐使用 文本格式**：
1. 打开 `题库_打印版.txt`
2. 打印到纸张
3. 简洁清晰，适合纸质阅读

### 在线分享
**可以使用 HTML 格式**：
- 可以直接上传到网站
- 或者通过邮件发送给家长
- 在任何浏览器中都能正常显示
