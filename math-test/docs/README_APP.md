# 📱 数学水平测试系统 - App 版本

## 🎉 已转换为 PWA 应用！

现在您可以像使用原生 App 一样使用这个系统了！

## ✨ 新增功能

### PWA 特性
- ✅ **添加到主屏幕**：在手机上像原生 App 一样使用
- ✅ **全屏体验**：无浏览器地址栏
- ✅ **离线支持**：缓存静态资源，快速启动
- ✅ **移动端优化**：完美的触摸体验

### 移动端优化
- ✅ 大按钮（符合移动端触摸标准）
- ✅ 流畅的滚动和动画
- ✅ 响应式布局（手机、平板、电脑完美适配）
- ✅ 优化的数学公式显示

## 🚀 快速开始

### 1. 启动服务

```bash
# 后端（终端1）
cd backend
python3 app.py

# 前端（终端2）
cd frontend
npm run dev
```

### 2. 在手机上访问

1. 确保手机和电脑在同一 WiFi 网络
2. 在手机浏览器中输入：`http://你的电脑IP:3000`
   - macOS: 在系统设置中查看 IP 地址
   - 或者使用 `ifconfig` 命令查看

### 3. 安装为 App

#### iOS (Safari)
1. 点击分享按钮
2. 选择"添加到主屏幕"
3. 完成！

#### Android (Chrome)
1. 点击菜单（三个点）
2. 选择"添加到主屏幕"
3. 完成！

详细说明请查看：[PWA_GUIDE.md](./PWA_GUIDE.md)

## 📦 构建生产版本

```bash
cd frontend
npm run build
```

构建后的文件在 `frontend/dist/` 目录，可以：
- 部署到任何静态网站托管服务
- 用户访问后即可安装为 App

## 🎯 使用场景

### 适合的场景
- ✅ 学校数学测试
- ✅ 学生自主学习评估
- ✅ 教师教学辅助
- ✅ 家长了解孩子数学水平

### 优势
- 📱 **随时随地**：手机、平板、电脑都能用
- 🚀 **快速启动**：添加到主屏幕后一键打开

## 📝 注意事项

1. **首次访问**需要网络连接
2. **API 请求**需要网络（测试功能）
3. **离线模式**下可以查看已缓存的页面
4. **iOS Safari** 需要使用 Safari 浏览器才能安装

## 🔄 更新应用

当有新版本时：
1. 用户刷新页面即可获取更新
2. Service Worker 会自动更新缓存
3. 无需重新安装
