# 🚀 GitHub Pages 部署指南

## 📋 概述

`题库.html` 是一个完全独立的交互式HTML文件，可以在GitHub Pages上直接部署，无需后端服务器。

## ✨ 功能特性

- ✅ **交互式答题**：点击选项进行答题
- ✅ **即时反馈**：答题后立即显示正确/错误判断
- ✅ **详细解析**：每道题都有详细的解析说明
- ✅ **知识点分析**：自动统计知识点掌握情况
- ✅ **能力评价**：基于IRT算法评估数学能力水平
- ✅ **学期表现**：分析不同学期的学习表现
- ✅ **学习建议**：根据答题情况提供个性化建议
- ✅ **数学公式渲染**：完美支持LaTeX数学公式

## 🔧 部署步骤

### 方法一：使用 GitHub Pages（推荐）

1. **创建 GitHub 仓库**
   ```bash
   # 在GitHub上创建一个新仓库，命名为 enjoymath-questions
   ```

2. **上传文件**
   ```bash
   # 将题库.html文件上传到仓库根目录
   git init
   git add 题库.html
   git commit -m "Add interactive question bank"
   git branch -M main
   git remote add origin https://github.com/你的用户名/enjoymath-questions.git
   git push -u origin main
   ```

3. **启用 GitHub Pages**
   - 进入仓库的 Settings
   - 找到 Pages 设置
   - Source 选择 `Deploy from a branch`
   - Branch 选择 `main` 和 `/ (root)`
   - 点击 Save

4. **访问网站**
   - 等待几分钟后，访问：`https://你的用户名.github.io/enjoymath-questions/题库.html`
   - 或者将 `题库.html` 重命名为 `index.html`，访问：`https://你的用户名.github.io/enjoymath-questions/`

### 方法二：直接使用 HTML 文件

1. **本地打开**
   - 双击 `题库.html` 文件
   - 或右键选择"用浏览器打开"

2. **上传到服务器**
   - 将 `题库.html` 上传到任何静态网站托管服务
   - 如：Netlify, Vercel, Cloudflare Pages 等

## 🎯 使用方法

### 答题流程

1. **开始答题**
   - 浏览题目，点击选项进行答题
   - 每道题只能答一次，答题后不能修改

2. **查看反馈**
   - 答题后立即显示正确/错误判断
   - 正确答案会用绿色高亮
   - 错误答案会用红色标记
   - 自动显示题目解析

3. **查看结果**
   - 完成部分或全部题目后
   - 点击"查看评价结果"按钮
   - 查看知识点掌握情况和能力评价

### 功能按钮

- **显示所有答案**：显示所有题目的正确答案（用于学习）
- **隐藏所有答案**：隐藏所有反馈信息
- **查看评价结果**：显示详细的分析报告
- **重置所有答题**：清除所有答题记录，重新开始

### 结果报告包含

- **总体能力水平**：基于IRT算法评估的能力等级
- **答题统计**：总题数、正确数、正确率
- **知识点掌握情况**：每个知识点的掌握度百分比
- **学期表现**：不同学期的学习表现分析
- **学习建议**：针对薄弱知识点的个性化建议

## 📊 功能说明

### 能力评价算法

系统使用简化的IRT（项目反应理论）算法：
- 根据答题正确率和题目难度估算能力值
- 能力值范围：-3 到 +3
- 能力等级：优秀 / 良好 / 中等 / 一般 / 需要加强

### 知识点分析

- 自动统计每个知识点的答题情况
- 计算掌握度百分比
- 识别薄弱知识点
- 提供针对性学习建议

### 学期表现

- 分析不同学期的学习表现
- 识别需要加强的学期内容
- 帮助制定学习计划

## 🎨 界面特点

- **Material Design 风格**：简洁现代的界面设计
- **响应式布局**：适配手机、平板、电脑
- **数学公式支持**：使用KaTeX完美渲染数学公式
- **交互友好**：清晰的视觉反馈和流畅的交互体验

## 💡 使用建议

1. **学习模式**：先自行答题，答错后再查看解析
2. **复习模式**：使用"显示所有答案"功能快速复习
3. **测试模式**：完整答题后查看评价结果，了解自己的水平
4. **针对训练**：根据评价结果，重点加强薄弱知识点

## 🔄 更新题库

如果题库有更新：

1. 运行导出脚本重新生成HTML：
   ```bash
   cd backend
   python3 export_interactive_questions.py
   ```

2. 将新的 `题库.html` 上传到GitHub仓库

3. GitHub Pages 会自动更新（可能需要几分钟）

## 📝 注意事项

- 所有数据存储在浏览器本地（localStorage），刷新页面会保留答题记录
- 如需清除所有数据，点击"重置所有答题"按钮
- 建议使用现代浏览器（Chrome, Firefox, Safari, Edge）以获得最佳体验
- 文件大小约 150-200KB，加载速度较快
