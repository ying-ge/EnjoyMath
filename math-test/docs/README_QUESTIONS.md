## 避免重复机制

系统已经实现了完善的去重机制：

1. **单次测试内去重**: 
   - 每次测试会话都会记录已答题目ID
   - 选择下一题时自动排除已答题目
   - 确保一次测试中不会出现重复题目

2. **跨会话去重**:
   - 不同测试会话独立管理
   - 每次新测试都可以选择所有题目
   - 但如果题库足够大，用户很难在多次测试中遇到完全相同的题目

## 扩充题库

### 添加新题目

编辑 `backend/expand_questions.py` 文件，在 `additional_questions` 列表中添加新题目：

```python
{
    'content': '题目内容（支持LaTeX公式）',
    'options': ['选项A', '选项B', '选项C', '选项D'],
    'correct_answer': 0,  # 正确答案索引（0-3）
    'difficulty': 0.0,    # 难度值（-3到3）
    'discrimination': 1.0, # 区分度（0.5-2.0）
    'knowledge_points': ['知识点1', '知识点2'],
    'category': '分类名称',
    'explanation': '题目解析（支持LaTeX）'
}
```

### 导入新题目

```bash
cd backend
python3 import_questions.py
```

### 查看统计

```bash
python3 question_stats.py
```
