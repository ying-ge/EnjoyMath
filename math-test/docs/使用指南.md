# 🎯 数学水平测试系统 - 使用指南

## 🚀 快速开始

### 1. 访问系统

**方式一：电脑浏览器**
- 打开浏览器，访问：**http://localhost:3000**

**方式二：手机浏览器（同一 WiFi 网络）**
1. 查看你的电脑 IP 地址：
   ```bash
   # macOS/Linux
   ifconfig | grep "inet " | grep -v 127.0.0.1
   
   # 或者使用
   ipconfig getifaddr en0  # macOS
   ```
2. 在手机浏览器输入：`http://你的电脑IP:3000`
   - 例如：`http://192.168.1.100:3000`

**方式三：安装为手机 App（PWA）**
1. 使用手机浏览器访问系统
2. **iOS（Safari）**：点击分享按钮 → "添加到主屏幕"
3. **Android（Chrome）**：点击菜单 → "添加到主屏幕"
4. 之后可以从主屏幕像 App 一样打开

### 2. 开始测试

1. **打开系统首页**
   - 会看到"数学水平测试系统"的欢迎界面

2. **输入学生 ID（可选）**
   - 可以输入你的学生 ID，或留空匿名测试
   - 输入 ID 可以跟踪你的测试历史

3. **点击"开始测试"**
   - 系统会自动为你选择第一道题目

4. **开始答题**
   - 认真阅读题目
   - 选择你认为正确的答案
   - 点击"提交答案"
   - 查看答案和解析

5. **继续答题**
   - 点击"下一题"继续
   - 系统会根据你的答题情况自动调整难度
   - 大约需要完成 15-20 道题目

6. **查看结果**
   - 测试完成后会自动跳转到结果页面
   - 查看你的能力水平、知识点掌握情况和学习建议

## 📱 详细步骤

### 步骤 1：访问系统

打开浏览器，在地址栏输入：
```
http://localhost:3000
```

如果服务正常运行，你会看到：
- 🎓 数学水平测试系统
- 系统介绍和功能特点
- "开始测试"按钮

### 步骤 2：开始新测试

1. **选择输入方式**
   - 输入学生 ID（可选，用于记录）
   - 或留空进行匿名测试

2. **点击"开始测试"**
   - 系统会创建新的测试会话
   - 自动选择第一道中等难度的题目

### 步骤 3：答题过程

**答题界面包含：**
- 📊 当前题目编号（如：第 1 题）
- 📈 进度条（显示测试进度）
- 📝 题目内容（支持数学公式）
- 🔘 选项按钮（A、B、C、D）

**答题步骤：**
1. 仔细阅读题目
2. 选择你认为正确的答案
3. 点击"提交答案"
4. 查看反馈：
   - ✅ 正确 ✓ 回答正确！
   - ❌ 错误 ✗ 回答错误
   - 📖 解析：查看详细解答
5. 点击"下一题"继续

**系统特点：**
- 🎯 **自适应难度**：根据你的答题情况自动调整
- 🧠 **能力评估**：实时估算你的数学能力水平
- 📚 **知识点覆盖**：涵盖初中数学所有主要知识点

### 步骤 4：查看测试结果

测试完成后，你会看到：

**总体能力水平**
- 优秀 / 良好 / 中等 / 一般 / 需要加强
- 具体能力值

**答题统计**
- 总题数
- 正确数
- 正确率

**知识点掌握情况**
- 每个知识点的掌握度百分比
- 各知识点的正确率

**个性化学习建议**
- 薄弱知识点提示
- 针对性的学习建议
- 优先级排序

## 💡 使用技巧

### 1. 最佳测试环境
- ✅ 安静的环境
- ✅ 充足的时间（每道题建议1-3分钟）
- ✅ 稳定的网络连接
- ✅ 认真思考后再作答

### 2. 理解自适应机制
- 系统会根据你的答题情况动态调整难度
- 如果答对较多，题目会变难
- 如果答错较多，题目会变简单
- 这是正常的，目的是准确评估你的能力

### 3. 查看解析
- 每道题提交后都会显示解析
- 认真阅读解析，理解解题思路
- 这有助于提高你的数学水平

### 4. 多次测试
- 可以多次进行测试
- 每次测试的题目会有所不同
- 题库有 85 道题目，足够支持多次测试

## 📊 测试参数说明

- **测试时长**：大约 15-20 道题目
- **预计时间**：15-30 分钟（取决于答题速度）
- **题目难度**：会根据你的能力自动调整
- **题库规模**：85 道题目，涵盖 17 个知识点

## ❓ 常见问题

### Q: 测试中途可以暂停吗？
A: 当前版本不支持暂停，建议一次完成。如果关闭页面，需要重新开始测试。

### Q: 可以重复测试吗？
A: 可以！每次测试都会创建新的会话，题目选择会不同。

### Q: 测试结果会保存吗？
A: 当前版本会保存在数据库中，但前端暂不支持查看历史记录。

### Q: 如何知道自己的进步？
A: 可以进行多次测试，对比不同时期的测试结果。

### Q: 题目太难/太简单怎么办？
A: 系统会自适应调整，如果感觉不合适，说明系统正在校准你的能力水平。
