## 🎯 学期映射规则

系统根据知识点自动判断学期：

### 七年级上
- 有理数
- 整式
- 一元一次方程

### 七年级下
- 实数
- 分式
- 二元一次方程组
- 不等式
- 平面几何

### 八年级上
- 二次根式
- 注意：2024年改版教材中，函数概念和一次函数已移至八年级下

### 八年级下
- 函数概念（2024年改版：从八年级上移至）
- 一次函数（2024年改版：从八年级上移至）
- 反比例函数
- 三角形
- 四边形

### 九年级上
- 一元二次方程
- 二次函数

### 九年级下
- 圆
- 统计
- 概率

## 📊 功能特性

### 题目显示
- 每道题目都会显示所属学期
- 学期标签以醒目的样式展示

### 结果分析
- 测试结果中包含学期表现分析
- 可以看到不同学期的正确率
- 帮助识别薄弱学期

## 🔧 查看学期统计

```bash
cd backend
python3 semester_stats.py
```

## 📝 添加新题目时

新题目会自动根据知识点分配学期，或可以手动指定：

```python
{
    'content': '题目内容',
    'knowledge_points': ['有理数'],  # 会自动分配为"七年级上"
    'semester': '七年级上',  # 或手动指定
    # ... 其他字段
}
```

## 🎓 使用场景

1. **学期针对性测试**：未来可以按学期筛选题目
2. **学期表现分析**：了解学生在不同学期的掌握情况
3. **学习进度跟踪**：识别哪些学期的内容需要加强
